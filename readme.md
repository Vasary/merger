# User merger
![GoLang](/doc/img/logo.png)

[![Build Status](https://travis-ci.org/Vasary/user_merger.svg?branch=master)](https://travis-ci.org/Vasary/user_merger)

```

DATABASE_PASSWORD=
DATABASE_NAME=
DATABASE_HOST=
DATABASE_PORT=
```
